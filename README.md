# ICRAAR website

Static website for **Ice Cold Refrigeration and Appliance Repair LLC**.

## Before publishing

Replace these placeholders in `index.html`:

- `(000) 000-0000`
- `tel:+10000000000`
- `service@icraar.com` if that address is not active
- `https://formspree.io/f/REPLACE_ME`

## Publish to GitHub Pages

Upload these files to the root of the `main` branch:

- `index.html`
- `styles.css`
- `script.js`

GitHub Pages will publish the site automatically after the commit.
